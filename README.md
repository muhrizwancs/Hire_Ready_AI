HireReady AI is a full-stack Flask web application that simulates real job interviews using AI. Users can upload their CV, get it analyzed, and practice mock interviews tailored to their profile.

🔥 Features
	•	User authentication (register/login/logout)
	•	CV upload (PDF, DOCX, TXT)
	•	AI-powered CV parsing
	•	Personalized mock interview sessions
	•	Real-time chat interface
	•	AI-generated feedback and scoring
	•	Interview history tracking

🛠️ Tech Stack
	•	Python (Flask)
	•	SQLite
	•	HTML, CSS
	•	Groq API (LLM)
	•	Gunicorn (production server)
